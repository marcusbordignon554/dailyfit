# 🏋️ DailyFit

Sistema de Gestão para Academias desenvolvido como projeto acadêmico da disciplina Introdução à Análise de Sistemas da FAETERJ.

## Objetivo

O sistema foi desenvolvido para auxiliar no gerenciamento de alunos, treinos, exercícios, dietas e suplementação.

## Funcionalidades

- Cadastro de usuários
- Cadastro de exercícios
- Cadastro de treinos
- Controle de dietas
- Suplementação
- Integração com IA Generativa

## Tecnologias

- GOOGLE AI STUDIO
- UML

## Engenharia de Software

- Levantamento de requisitos
- Documento de requisitos
- Casos de uso
- Diagrama de Classes
- Documentação técnica
